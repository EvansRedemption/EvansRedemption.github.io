反思我写的面试题

发表于 2019-04-03 | 分类于 交互面试题试答 | | 阅读次数:

为什么我总是写不出我想要写的东西，反思。

今天讨论面试题前，先跑个火车，把我最近的一些想法说一说。

其实在选择这道题之前，我已经选择过2道题，尝试进行回答，但很失败的是我根本写不下去。
每一题都是似是而非的感觉，总觉得自己能说点什么，但又说不出来。
问题出在哪里，反思以后，我认为在三个方面。

其一，写文章和真实面试直接口述回答的差别。
写文章的时候希望自己能够条理清晰，分门别类的从各个角度去阐述，并且希望每个角度都尽善尽美，彻底说到点上。
而面试的时候可能会更偏向于零散式的说一些自己get的点，也会有一个脉络在，但不会在某个主题上硬扛，想到多少说多少。

其二，对于问题，或者说这些专业知识没有一个脉络。
比如现在这道问题，显然就应该从社交体系、用户研究的脉络去回答，社交体系包含社交背景、发展balabala，我根本回答不全。
这一点无论是写作还是面试，对我而言都是巨大的硬伤。

其三，我不懂的东西太多了，很多东西都想不到一个合适切入点，或者切得太表面，没有足够的专业积累。

第二第三个问题是我最大的问题，所以我现在准备这么做。
以口述的标准回答问题，然后立刻去做相关搜索并学习，最终产出一篇正文形式的内容。
