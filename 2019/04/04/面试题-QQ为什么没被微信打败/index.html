面试题-QQ为什么没被微信打败

发表于 2019-04-04 | 分类于 交互面试题试答 | | 阅读次数:

QQ现在是仅次于微信的APP，你觉得没有被微信打败的原因是什么？

面试口语化回答

根本因素

首先从根本原因说起，很简单，对于不同的用户人群，使用场景，商业目的而言，根本不存在打败不打败的说法，只能说两者适用情况不同。
说个思想实验，如果微信和QQ两者是竞争关系，那腾讯从商业目的角度就不可能让两者并行这么久。

我们说有没有打败，一个很简单的衡量因素就是有没有人用了，市场占有率怎么样，商业营收怎么样。
QQ在微信诞生之前就积累的海量的用户，微信的壮大在最初很大一部分都是因为QQ的导流。所以并不存在QQ没有人用的情况。
而且，QQ和微信的用户人群在很大程度上是重叠的，即便是重度的微信使用者，他也不会抛弃QQ，因为他有一部分关系链沉淀在QQ里。

用户角度

QQ和微信都是IM社交产品，我们使用IM并不单单因为某个软件好用而跳转过去使用，而是“我周围的人在用哪个IM，决定了我去用哪个”
打个比方，现在很多人说TG比微信好用，我也确实去体验了一下，很多功能确实都比微信好，甚至说碾压都不为过，但我不会去用，为什么，因为我周围没人用。我一个人跑去用TG，和机器人聊天吗？

再说回用户人群，我记得QQ和微信在发展战略上有一个很大的差异化。QQ是年轻化，微信是工具化。工具化可以理解为用完即走，和时间不是那么充裕的白领、成年人等用户相契合。
那么，我是不是可以这么理解。QQ注重未进入社会的人群，微信注重已经踏入社会的人群。

先说说为什么会有这样的发展战略，QQ我想不太清楚，可能与它的增值服务角度相关，暂且不表。
而微信的话，我是这样猜测的，微信这个产品在最开始可以理解为替代了移动时代的短信功能，所以它的部分用户群就是对PC不是那么精通的社会人。微信只要一个手机，一个手机号码就能注册使用，对于他们更方便使用。
年轻人总会成长，踏入社会，被老一辈人带进微信。同时成长起来的年轻人呼应我前面说到的一部分用户的关系链沉淀在QQ里，所以形成了QQ微信双持的情况。
这个点是我的猜测，纯粹带一句而已。

功能角度

再从功能角度去说说。
“每天有一万个人教我怎么做产品”——张小龙
产品总有优缺点，有些功能就是QQ比微信好，比如消息同步，比如文件传输，比如群功能那个。
微信也有比QQ好的，比如专注聊天核心功能，比如小程序等等。
有些人认为QQ臃肿，微信克制。但实际上你真的去梳理两者的的信息架构，你甚至会发现微信的功能比QQ还要多。只是微信更突显核心功能，QQ将功能平铺导致的认知错觉而已。
这些只是QQ为了满足年轻用户对趣味性和娱乐性的需求，微信为了满足工具产品的认知简单而展现的不同形态而已。
两个形态并不完全重叠产品，本就有不同的发展角度，说不上谁打败了谁。

票圈讨论

在票圈发了条讨论贴，收获了一些妖刀和清奇的思路，放出来分享一下，和我观点相同的就不做赘述了。

@金木研

最大原因是微信需要身份证认证和手机认证，大多未成年没有自己的手机，就算有也是父母用过了的基本都认证过微信，认证起来不方便，所以只能选择QQ

从另一个角度去阐述了QQ和微信用户群体不同的一个点，对比我说的猜测更优。

@顾静海

因为社会再部落化了，很多人选A的时候就必然有人要选B。甚至同一个人，在微信人生以外也需要另一个场景做自己。facebook这么强势，川普还是喜欢发twitter。功能本质没什么区别，就是不一样而已
都说QQ低龄化，但QQ的低龄化可能就是微信成年化的产物。未进入社会的年轻人集中在QQ上，使用成年人不理解的语言，目的就是远离成年人的微信世界。
淘宝天猫为了他们的“最大目标客户群体”做出的努力，就是催生京东和拼多多的原因。
当一个社群达到一定人员规模量级的时候，就会自然产生宗教化的身份认同。买京东的人会说自己追求的是用户体验，买拼多多的人会强调性价比才是硬道理。其实本质上和上淘宝没差别，很多商品也是相同的供应商。但再部落化行成的鄙视链，给陌生人社会里的个体找到了归属感。
腾讯在社交上命好一点，有两个产品收割不同部落，总比一个产品防御别人进攻来得安心。

角度相当刁钻，至少对我而言是这样的，我不了解社会再部落化这个概念。
这一条和金木研的合并起来看，倒像是硬币一体两面，阐述了微信和QQ不同用户人群产生的原因，很有意思。我也需要去注重这种一体两面的倒推思维。
然后，“当一个社群达到一定人员规模量级的时候”让我有一种熟悉的感觉，居斯塔夫在《乌合之众》中提出的概念，是不是正好就是这块内容的解释呢？看来是时候重新去读一读了。
这条评论，就是我认为我缺失的高纬度视野，有种提纲挈领的感觉。从抽象角度去描述一个具体现象，并且还能让人看得懂。学习了。
